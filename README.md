## Features

- **6 Weapons**: Knife, Pistol, SMG, Shotgun, AR15, Minigun
- **3 Enemy Types**: Fast melee minions, heavy tanks, ranged shooters
- **3 Maps**: Easy (outdoor arena), Medium (industrial), Hard (hellish caverns)
- **Wave System**: 5 waves per map with progressive difficulty
- **Full UI**: Animated menus, HUD, pause, game over, victory screens

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Space | Jump |
| Shift | Sprint |
| Left Click | Fire |
| 1-6 | Switch weapons |
| R | Reload |
| ESC | Pause |

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **Three.js** - 3D rendering
- **Vite** - Build tool
- **Web Audio API** - Procedural sound effects
